SELECT:

It is used to retrieve the data from the table

SYNTAX:

SELECT COLUMNAME FROM TABLENAME

EXAMPLE:

select name from das;--->single column
select name,age from das;--->multiple columns
select * from das ---->all columns




distinct:

it is used to fetch different values from the table

syntax:
select distinct(columname) from tablename

example:
select distinct(location) from emp






where 

it is used to filter the records based on specific condition

SYNTAX:
select columname from tablename where condition;

EXAMPLE:

select * from das where name="hari";

AND:

select * from das where location="Chennai" and name="hari"

OR:
select * from das where location="Chennai" and name="hari"


IN OPERATOR:
 
it is used to fetch the records based on multiple values

SYNTAX:

select columname from tablename where columnname in(v1,v2..,)
select columname from tablename where columnname not in(v1,v2..,)

select * from das where location in ("Madurai","Chennai","salem")


BETWEEN

it is used to fetch the records based on certain range

SYNTAX:

select columname from tablename where columname between v1 and v2;

EXAMPLE:

select * from das where salary between 1000 and 5000;

select * from das where salary not between 1000 and 5000;


ISNULL


It is used to fetch the records with null values

SYNTAX:

Select columname from tablename where columname is null;

EXAMPLE:

select * from tablename where salary is null;

select * from tablename where salary is not null;


LIKE :
it is used to fetch the records based on specific pattern matching

wildcards:

   %  - any chr any no of time
   _  - any chr only one time


s% - first chr s

%s - last chr s

_s% - second chr s

%s_ - last before chr s

__s% - third chr s

%s% - middle s

%s_s% -alternative s

%ss% - continuous s



SYNTAX:

select columname from table where columname like 'pattern'

EXAMPLE:

select name from das where name like 'm%';

select name from das where name not like 'm%';

ORDER BY:

It is used to sort the results in ascending or descending order.
The default order method is ascending order.

SYNTAX:

select columname from tablename order by columnname asc/desc;


EXAMPLE:

select * from employee order by salary desc;


AGGREGATE FUNCTIONS:

It performs with multiple values in a columns and returns the single output

sum() - returns total values
min() - returns minimum value
max() -returns maximum value
avg() -returns mean value
count() -returns number of value

SYNTAX:

select agg(columname) from tablename where condition.

EXAMPLE:

select count(*) from employee:

GROUP BY:

It is used to grouping the records based on same values in a column.
It is always works with aggregate functions.

SYNTAX:
select agg(columname) from tablename where condition group by columname;

EXAMPLE: Find total salary of each location.

select sum(salary),location from employee group by location;

Find total salary of each location whose dep is hr

select sum(salary),location from employee where dep="HR" group by location having sum(salry) >5000;

HAVING:

It is used to filter the group by records.

it always works with group by clause

SYNTAX:
select agg(columname) from tablename where condition group by columname having condition

EXAMPLE:
find total salary of hr in each location whose total salary is greater than 500
   
select sum(sal) from emp where pos='hr" group by location having sum(sal) >500;


find number of employees in each dep,which ep has more than 3 employees
   
select count(name) from emp group by dep having count(name)>3;

LIMIT

it is used to fetch the number of rows from the table

SYNTAX
Select columnname from tablename limit no.of.rows

EXAMPLE:
to fetch first 10 rows
select name from emp limit 10;

to find top 10 salary in table
select * from emp order by salary desc limit 1;

OFFSET;

it is used to skip the number of rows
it always work with limit

SYNTAX:

select columname from tablename limit no.of.rows offset no.of.rows

Example:
find fifth max salary salary
select salary from emp order by salary desc limit 1 offset 4;

ALTER Command in MySQL Workbench

The ALTER command is used to modify an existing table structure without deleting the table.

1️⃣ Add a New Column
ALTER TABLE table_name
ADD column_name datatype;


Example:

ALTER TABLE employees
ADD salary INT;

2️⃣ Add Multiple Columns
ALTER TABLE table_name
ADD column1 datatype,
ADD column2 datatype;

3️⃣ Modify Column Datatype / Size
ALTER TABLE table_name
MODIFY column_name new_datatype;


Example:

ALTER TABLE employees
MODIFY salary DECIMAL(10,2);

4️⃣ Change Column Name
ALTER TABLE table_name
CHANGE old_column new_column datatype;


Example:

ALTER TABLE employees
CHANGE name full_name VARCHAR(50);

5️⃣ Rename Table
ALTER TABLE old_table_name
RENAME TO new_table_name;

6️⃣ Drop (Delete) a Column
ALTER TABLE table_name
DROP column_name;
